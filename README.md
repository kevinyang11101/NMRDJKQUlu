# 停车场管理系统 java64

## 项目概述

停车场管理系统是一款基于Web的停车场管理软件，采用前后端分离的架构设计，前端使用Vue.js框架，后端基于SpringBoot框架，并且整合了Mybatis-Plus作为数据访问层。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis-Plus
  - Vue

## 功能模块

### 系统角色

- **普通用户**: 可以登录系统进行以下操作
  - 个人信息管理
  - 预定停车位
  - 缴费信息管理

- **管理员用户**: 可以登录系统进行以下操作
  - 用户信息管理
  - 车辆信息管理
  - 停泊车辆管理
  - 车辆进出管理
